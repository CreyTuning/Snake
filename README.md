# Snake

## Simple snake-game made with C/C++.


### Start menu

1) Start the game (run the game)

2) Difficulty level (select the sifficulty of game)


### Difficulty screen
Four difficulty levels

1) Facil (easy)

2) Normal

3) Dificil (Hard, I recommend this)

4) Modo destrucción (IMPOSSIBLE)
